Práctico 2 — Posicionamiento

static • relative • absolute • fixed • sticky
Static (por defecto) — los elementos siguen el flujo normal del documento.

Este párrafo está en posición static (valor por defecto). No se ha aplicado ninguna propiedad top/right/bottom/left sobre él.

Relative — se desplaza respecto a su posición normal, pero sigue ocupando su espacio original en el flujo.

Contenedor (position: relative)

Este elemento tiene position: relative y se ha movido left:20px; top:8px;. El espacio original se mantiene.
Absolute — se posiciona respecto al ancestro posicionado más cercano (ej. algún ancestro con position distinto a static).

Contenedor con position: relative (referencia para el absolute)

Soy position: absolute — estoy anclado a la esquina inferior derecha del contenedor.
Sticky — se comporta como relative hasta que la página se desplaza hasta un umbral; entonces se fija como si fuera position:fixed dentro de su contenedor.

Artículo largo (para hacer scroll)

Repite texto para que haya scroll y ver el comportamiento sticky:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Reprehenderit, illum. (repite varias veces)...

(espacio vertical extra para poder desplazar)

Tarjeta position: sticky; top:80px;

Cuando haces scroll, esta tarjeta se pega a la parte superior de la ventana hasta que el contenedor deje de tener espacio.

Fixed — fijado respecto a la ventana del navegador; no se mueve al hacer scroll (ej. header o footer).

El encabezado superior (con fondo colorido) y el pie de página están position: fixed — siempre visibles.

Fragmento de ejemplo del código usado para la tarjeta sticky:

<div class="sticky-card" style="position:sticky; top:80px;">Contenido pegajoso</div>
